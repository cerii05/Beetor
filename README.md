# Beetor

About 
BeeTor adalah sebuah website yang bertujuan untuk memudahkan mahasiswa Binus University dari jurusan manapun dalam menemukan jasa tutoring oleh tutor pilihan dan berkualitas untuk mendapatkan pemahaman lebih mengenai materi perkuliahan. Inspirasi kami dalam mendirikan website ini adalah pengalaman pribadi sebagai mahasiswa yang suatu saat dapat merasa kesulitan dalam memahami materi perkuliahan.
